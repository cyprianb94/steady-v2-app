06 · Multiple new runs

  • When shown: strava.sync returned more than one new activity not yet linked to a session. Common when users sync less frequently.
  • Recommended card: the one that matches today's session gets the forest border, filled ► icon, and MATCHES TODAY tag. All other cards are calm/slate.
  • Metrics line is one mono row so rows align visually — the eye can scan distance and pace down the column.
  • Timestamps on the right side are relative ("06:58" = today, "Apr 9 · 19:20" = earlier). Uses Space Mono so they column-align across rows.
  • Empty-state hint at the bottom handles the "Strava hasn't finished processing the upload yet" case — suggests pull-to-refresh.
  • Tap behaviour: tapping any run → opens that activity in Run Detail (screen 03 or 04 depending on match).
