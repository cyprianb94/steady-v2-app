Home colour-language pilot

High-fidelity wireframe based on the supplied Home screenshot. This pass keeps the existing information architecture and tests semantic colour on Today and This week only.

Language rules

The goal is faster scanning, not a more colourful Home tab.

Pilot scope

Today and This week are the proposal. Weekly volume is shown for context, with only the obvious distance-token correction in the collapsed card.

Today card

The session type moves into the chip. The card border and headline stay neutral. Distance, pace, heart rate, and feel carry the metric colours.

This week rows

Each row keeps a small session dot. Status icons carry judgement. Actual distance can use cobalt; future planned values stay muted.

Guardrail

A row should not carry more than two strong colour signals. If status needs attention, metric colour yields first.

Metric tokens used
Distance / volume #3D55A4
Pace / speed #187F7A
Heart rate #BD433B
Effort / feel #765098

Intent: remove the current ambiguity where interval clay colours the whole Today card and off-target amber colours distance values. The value colour should say what the number is; the status icon should say how it went.
