Run detail colour-language prototype

High-fidelity wireframe applying the proposed Steady semantic colour system. The important shift: cards stay parchment, labels stay quiet, and the values/bars carry the metric language.

Language rules

This frame deliberately separates phase, session, metric, status, and action colour.

Core rule

Phase frames the block. Session identifies the workout. Metric identifies the number. Status interprets the result. Action tells you what can be done.

Design intent

The screen should become more scannable without becoming bright. The user should recognise distance, time, pace, and heart rate before reading the labels.
