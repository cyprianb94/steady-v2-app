Small interval correction

Keep the product move small: Home gets a corrected summary, Run Detail gets enough interval context to explain it. No new analytics dashboard, no extra chart surface, and no whole-run average pace driving interval verdicts.

Minimal scope

The fix is not "add analytics everywhere". The fix is "grade the right part of the run".

Home verdict
Use quality-work analysis first.

If all 400m reps are inside range, the headline is On target. Home shows only total distance, reps in range, rep average, and interval BPM.

Run detail
Add one modest interval summary.

Show work-rep stats above the existing splits. Do not introduce a new chart-led analytics screen.

Linear scope
STV2-185 + STV2-67.

Use interval HR and elapsed split time. Leave HR drift, load breakdowns, and Steady commentary out of this pass.

Fallback
No structured laps, no fake grade.

When we only have kilometre splits, show total run data and a neutral prompt to inspect splits. Do not say Eased off.

Clay marks interval work and target-band analysis.

Forest marks the completion verdict.

Amber marks recoveries and secondary watch points.
