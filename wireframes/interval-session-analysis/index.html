Structured quality summaries

Keep the product move small: Run Detail gets one shared Quality Summary component for interval and tempo sessions. The Home/TODAY card is out of scope and will be handled in its own redesign.

Scoped project

The fix is not "add analytics everywhere". The fix is "grade the right part of the run" with one reusable component.

Out of scope
Leave the TODAY card alone.

This project does not redesign Home. It only gives Run Detail a reliable structured-session summary.

Run detail
One component, two variants.

`Interval summary` and `Tempo summary` share structure. Session accent changes, metric treatment stays consistent.

Linear scope
STV2-192, STV2-194, STV2-195.

Domain summary, shared Run Detail card, and target-aware Splits. Existing STV2-185/STV2-67 are linked.

Fallback
No structured laps, no fake grade.

When we only have kilometre splits, show total run data and a neutral prompt to inspect splits. Do not say Eased off.

Clay marks interval work and target-band analysis.

Forest marks the completion verdict.

Amber marks tempo session identity.
