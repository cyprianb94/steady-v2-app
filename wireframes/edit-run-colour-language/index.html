Edit run colour language

Static wireframes based on the current edit-session screenshots. The proposed shift: session colour identifies the run type, metric colour becomes a quieter row accent, and clay stays reserved for the primary save action.

Session

Header type words and session-type chips keep easy, interval, tempo, long, and rest colours.

Metric

Distance, pace, and duration use colour as a small row cue, not a filled control system.

Pace UX

Expanded pace rows show training paces first, then manual pace chips below.

Action

Cancel is neutral. The sticky save button remains clay as the primary action.
