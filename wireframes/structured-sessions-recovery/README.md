# Structured Sessions and Recovery Runs - Wireframes

Static high-fidelity exploration for the structured sessions and recovery run PRD.

Open `index.html` directly in a browser. No build step is required. A rendered preview is generated as `structured-sessions-recovery.png`.

## Screens

| # | Frame | Purpose |
|---|---|---|
| 01 | Edit session | Adds Recovery, Plan note, and Run structure into the current session editor layout |
| 02 | Design your week | Adds Recovery and structure indicators inside the current plan-builder weekly layout |
| 03 | Review your block | Adds structure summaries and indicators inside the current Review your block Weeks tab |
| 04 | Home / block today | Shows how a structured long run appears in the current Home surface |
| 05 | Run structure | New full-screen editor for segments, repeat groups, totals, and plan note |

## Design intent

- Steady remains bring-your-own-plan.
- Session role colour stays primary.
- Recovery run colour is related to phase recovery purple but distinct from it.
- Run structure and plan note indicators stay small and quiet in existing week rows.
- Segment rows avoid a rainbow treatment; numeric values use metric colour only where helpful.
- The simple editor remains the default path.
